## K-Nearest Neighbours (KNN)

–î–ª—è –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–π —Ç–æ—á–∫–∏ $x$ –∏ –æ–±—É—á–∞—é—â–µ–≥–æ –Ω–∞–±–æ—Ä–∞

—Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –¥–æ –∫–∞–∂–¥–æ–π —Ç–æ—á–∫–∏ –Ω–∞–±–æ—Ä–∞ –∏–∑–º–µ—Ä—è–µ—Ç—Å—è –ø—Ä–∏ –ø–æ–º–æ—â–∏ **–µ–≤–∫–ª–∏–¥–æ–≤–æ–π –º–µ—Ç—Ä–∏–∫–∏**:

### üîπ KNN –ö–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ü–∏—è
